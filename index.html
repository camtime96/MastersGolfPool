Shared scores.json source

Every user should open the same HTML page. That page should load the same public scores.json URL from GitHub Pages. When you update the JSON file in GitHub and commit it, everyone refreshes to the same numbers.

What changes are shared?

Manual edits on this page are local only until you publish the generated JSON into scores.json in GitHub.
Shared to everyone:
• anything saved in the repo's scores.json
Not shared automatically:
• edits you make only inside your current browser tab
